Split or Steal: The Prisoner's Dilemma
Developed By: Lauren Dickman, Courtney Arrowood, Dale Donnelly

https://github.com/LDickman/CS260S_Prisoner_Dilemma

An educational game meant to provide an interactive way to learn the game theory and strategies
of the iterated prisoner's dilemma.

Developed as part of an educational game development class, Hanover College 2022
